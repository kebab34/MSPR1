## 1. Vue d'Ensemble

Ce document présente le diagramme complet des flux de données du système HealthAI Coach, depuis la collecte jusqu'à la visualisation.

## 3. Flux Détaillé : Pipeline ETL

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         SCHEDULER (APScheduler)                              │
│                         Cron: 0 */6 * * * (toutes les 6h)                    │
└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    │ Déclenchement
                                    ▼
    ┌─────────────────────────────────────────────────────────────────┐
    │                    run_etl_pipeline()                            │
    │                                                                  │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  1. EXTRACT EXERCISES                                    │  │
    │  │     extract_exercises_from_exercisedb()                  │  │
    │  │     → DataFrame pandas                                    │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │                           ▼                                      │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  2. TRANSFORM EXERCISES                                  │  │
    │  │     transform_exercises_from_exercisedb()               │  │
    │  │     clean_data()                                         │  │
    │  │     validate_data()                                      │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │                           ▼                                      │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  3. LOAD EXERCISES                                       │  │
    │  │     loader.upsert_dataframe("exercices")                 │  │
    │  │     → Supabase                                           │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  4. EXTRACT FOODS                                        │  │
    │  │     extract_from_csv() (si fichiers disponibles)         │  │
    │  │     → DataFrame pandas                                    │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │                           ▼                                      │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  5. TRANSFORM FOODS                                       │  │
    │  │     transform_foods_from_csv()                            │  │
    │  │     clean_data()                                           │  │
    │  │     validate_data()                                        │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │                           ▼                                      │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  6. LOAD FOODS                                           │  │
    │  │     loader.upsert_dataframe("aliments")                  │  │
    │  │     → Supabase                                           │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │  ┌──────────────────────────────────────────────────────────┐  │
    │  │  7. GENERATE TEST USERS                                  │  │
    │  │     (si utilisateurs manquants)                         │  │
    │  │     → Supabase                                           │  │
    │  └──────────────────────────────────────────────────────────┘  │
    │                           │                                      │
    │                           ▼                                      │
    │                    Logging & Rapport                             │
    └─────────────────────────────────────────────────────────────────┘
```

## 7. Légende

- **Flèches pleines** : Flux de données principal
- **Flèches pointillées** : Flux de contrôle/erreur
- **Rectangles** : Composants système
- **Parallélogrammes** : Données/Formats
- **Losanges** : Décisions/Validations
